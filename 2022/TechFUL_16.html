「Bomb on Grid」 / 難易度 : 5
問題タイプ:コーディング問題 目標タイム:50分 アルゴリズム/ データ処理

問題文

縦 [Math Processing Error]105 マス、横 [Math Processing Error]105 マスのグリッドの上に [Math Processing Error]N 個の爆弾が置かれています。
[Math Processing Error]i 個目の爆弾は左から [Math Processing Error]Xi マス目、上から [Math Processing Error]Yi マス目にあるマスに置かれています。
爆弾は爆発すると同じ行または同じ列にある爆弾を誘爆させます。この効果は連鎖します。
あなたはいくつかの爆弾を選んで爆発させることで、[Math Processing Error]N 個の爆弾をすべて爆発させたいです。
最小でいくつの爆弾を選ぶ必要があるか求めてください。

入力される値

N
X_1 Y_1
X_2 Y_2
...
X_N Y_N
  • [Math Processing Error]1 行目に整数 [Math Processing Error]N が与えられます。
  • [Math Processing Error]2 行目から [Math Processing Error]N+1 行目に整数 [Math Processing Error]Xi,Yi が与えられます。

期待される出力値

答えを [Math Processing Error]1 行で出力してください。

制約

  • 入力はすべて整数である。
  • [Math Processing Error]1≤N≤105
  • [Math Processing Error]1≤Xi,Yi≤105
  • [Math Processing Error]i≠j ならば [Math Processing Error](Xi,Yi)≠(Xj,Yj)
